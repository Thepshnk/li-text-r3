Это одна из самых легких текстовых моделей на основе Li Text, которая доступна для свободного использования. Модель называется Li Text R3.

Для работы с ней вам потребуются следующие зависимости:
`pip install numpy scikit-learn nltk`
Обратите внимание, что для корректной работы нейросети и базы данных они должны находиться в одной директории.
